# SpotClone
A sleek Spotify clone with music playback, modern UI, and responsive design.
A frontend Spotify-inspired music player built with HTML, CSS, and JavaScript.

## ✨ Features

- 🎧 Play / Pause songs
- ⏩ Progress bar with seek support
- 🔍 Search songs by name
- 🎨 Now playing highlight on song card
- 📱 Responsive design (mobile friendly)
- 👨‍🎤 Popular Artists section

## 🛠️ Tech Stack

| Technology | Use |
|------------|-----|
| HTML5 | Structure |
| CSS3 | Styling & Layout |
| JavaScript | Logic & Audio API |

